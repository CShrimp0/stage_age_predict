# results 目录说明

推荐优先阅读：

- `reports/single_ml/`
  - 单个 ML 实验对比多条 `bio_age` 轴的清晰报告。
  - 这是当前仓库最推荐直接查看的结果入口。

原始计算输出：

- `bio_age_feature_benchmark/`
  - `bio_age` 参考轴拟合的原始结果目录。
  - 包含完整 `feature_sets/`、leaderboard、subject diagnostics 和原始图。

- `ml_vs_bio_age/`
  - 多个 ML runs 对比多条 `bio_age` 轴的原始结果目录。
  - 这是进阶分析入口，不是当前最推荐的首读结果。

历史样例：

- `archive/`
  - 旧结果迁移过来的样例，保留作参考。
  - 不作为当前主报告入口。
